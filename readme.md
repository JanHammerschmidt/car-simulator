## Get it running

### Requirements

* node (and npm)
* bower
* python (or some other way to serve a directory via http)
* You need the models and textures, which are not in this repository.

### Build

* `node install`
* `bower install`
* `webpack`

### Run 

* `python -m http.server 8080`
* access `http://localhost:8080/main.html`
